## What is Scroll Snap? <a name="what-is-scroll-snap?"></a>

Scroll snapping is a CSS technique that allows customizable scrolling experiences.
This gives us a way to scroll to certain points as the user scrolls through a document.
This can be helpful in creating a more app-like experience on mobile.

## Scroll Snapping in Tailwind <a name="scroll-snapping-in-tailwind"></a>

Utility classes for scroll snapping were added in TailwindCSS v3  
With these classes we can define things like scroll snap type, alignment, whether the scrolling will always be stopped on an element, padding and margin for items.

### Scroll Snap Type <a name="scroll-snap-type"></a>

Define the orientation of the snap container and how strict it is on the elements.

- These classes can only be applied to scrolling containers.

| Class            | Description                                                 |
| ---------------- | ----------------------------------------------------------- |
| `snap-x`         | enable horizontal scroll snapping                           |
| `snap-y`         | enable vertical scroll snapping                             |
| `snap-mandatory` | force a snap container to always come to rest on an element |
| `snap-proximity` | don't force a snap container to always rest on an element   |
| `snap-both`      | enable horizontal as well as vertical scroll snapping       |
| `snap-none`      | disable scroll snapping                                     |

### Scroll Snap Align <a name="scroll-snap-align"></a>

These classes are used to specify the alignment of elements in a container.
To put it in another way: how the item is positioned when snapped.

- Apply these classes to elements inside a container

| Class         | Description                              |
| ------------- | ---------------------------------------- |
| `snap-center` | align in the center of the container     |
| `snap-start`  | align on the left side of the container  |
| `snap-end`    | align on the right side of the container |

### Scroll Snap Stop <a name="scroll-snap-stop"></a>

Define whether the scrolling should always be stopped on an element or not.

- Apply these classes to elements inside a container

| Class         | Description                                      |
| ------------- | ------------------------------------------------ |
| `snap-always` | always stop on an element while scrolling        |
| `snap-normal` | allow scrolling to be stopped outside an element |

### Snap Padding <a name="snap-padding"></a>

Define the offset of an element within a snap container.
You can use it like: `scroll-` followed by the usual padding class.
For Example: `scroll-px-3`

### Snap Margin <a name="snap-margin"></a>

Define gaps around elements in a snap container.
You can use it like: `scroll-` followed by the usual margin classes.
For Example: `scroll-ml-3`
